# setup-fd

Set up the [`fd`] command (aka [fd-find][`fd`]) in Github Actions.

## Usage

Select the most recent version:

```yaml
- uses: coriolinus/setup-fd
```

### Select a specific version

```yaml
- uses: coriolinus/setup-fd
  with:
    version: v10.0.0
```
